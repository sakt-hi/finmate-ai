# FinMate AI

AI-powered WhatsApp finance assistant.

## Features

- Expense tracking
- Multi-expense parsing
- AI summaries
- Conversational analytics
- Budgeting
- Confirmation workflows
- Stateful conversations

## Stack

- FastAPI
- Groq
- SQLite
- Twilio
- Python
